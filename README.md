## ğŸ“¦ Installation

### Prerequisites
- Node.js 18+
- MongoDB (local or Atlas)
- Gemini API Key

### Setup

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd FitMaster
   ```

2. **Backend Setup**
   ```bash
   cd server
   npm install
   ```
   
   Create `.env` file:
   ```
   PORT=5000
   MONGO_URI=mongodb://localhost:27017/fitmaster
   JWT_SECRET=your-secret-key
   GEMINI_API_KEY=your-gemini-key
   ```

3. **Frontend Setup**
   ```bash
   cd client
   npm install
   ```

4. **Run Development Servers**
   
   Terminal 1 (Backend):
   ```bash
   cd server
   npm run dev
   ```
   
   Terminal 2 (Frontend):
   ```bash
   cd client
   npm run dev
   ```

5. **Access the App**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:5000
